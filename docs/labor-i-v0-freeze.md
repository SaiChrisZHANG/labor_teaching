# Labor I version 0 freeze

## What version 0 means

Version 0 is the pre-editorial backbone of the course. It is intended to preserve the structure, workflow, and first full set of weekly source packs before a full authorial revision pass.

## What should be frozen

- the Labor I outline and landing page
- all Week 1–13 source packs
- the shared bibliography as currently assembled
- workflow docs, prompts, and validation conventions
- current chapter/slide/lab drafts produced under the standardized workflow

## What is explicitly *not* implied by version 0

- final editorial voice
- final paper selection in every lab
- final figures and tables
- final slide style
- final course pruning for an actual teaching term

## Why the freeze is useful

- preserves a clean job-market-ready milestone
- creates a stable base for editorial revision
- separates workflow validation from content refinement
- makes it easier to start Labor II without destabilizing Labor I

## Suggested git markers

- branch: `main`
- annotated tag: `labor-i-v0`
- GitHub release title: `Labor I v0`
