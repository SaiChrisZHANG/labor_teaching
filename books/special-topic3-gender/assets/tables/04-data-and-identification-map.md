# Data and identification map

| Data source | Best for | Typical methods | Strength | Main limitation |
|---|---|---|---|---|
| Audit / screening data | hiring-stage bias and screening rules | randomized CV signals, blind-screen designs | clean entry-margin identification | limited downstream outcomes |
| Personnel records | promotions, evaluations, authority, task allocation | event studies, hazard models, supervisor/manager designs | rich internal margins | one-firm external-validity concern |
| Matched employer-employee data | firm premiums, sorting, bargaining | AKM-style decompositions, rent-sharing, mobility designs | clean firm-vs-worker separation | mechanism often indirect |
| Survey-linked admin data | beliefs, informal interactions, mentoring, schmoozing | quasi-experimental assignment + survey measurement | can observe hidden channels | measurement error / external validity |
| Firm policy reforms | transparency, equal-pay compliance, disclosure | diff-in-diff, event study, triple-diff | policy relevance | concurrent firm adaptation complicates interpretation |
| Workplace-condition surveys + admin | harassment, retention, segregation | transition models, valuation exercises, linked panels | captures nonwage conditions | measurement and reporting issues |
