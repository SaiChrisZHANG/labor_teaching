# Transfer exercise

The transfer path uses a synthetic job-ladder dataset inspired by `@haltiwangerHyattKahnMcEntarfer2018`.

The transfer object is intentionally different from the reproduction object:

- reproduction: U-to-E hazard and composition-adjusted job finding;
- transfer: E-to-E mobility, upgrading, and job ladders across market states.

This contrast is the whole pedagogical point of Week 4.
