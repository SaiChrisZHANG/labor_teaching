# Transfer data notes

The transfer file records employment stocks and job-to-job transitions for workers starting in low-, middle-, and high-wage firms under two market states:

- `tight`
- `loose`

For each origin tier, the file contains:

- the number of employed workers,
- the number of E-to-E moves,
- the number of upward moves to a better-paying tier.

Students should interpret this file as a synthetic ladder summary, not as a public-use administrative extract.
