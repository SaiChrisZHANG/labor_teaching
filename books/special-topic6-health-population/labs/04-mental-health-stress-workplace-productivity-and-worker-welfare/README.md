# Week 4 Lab: Mental Health, Stress, Workplace Productivity, And Worker Welfare

This lab is the bounded Week 4 teaching path for Labor Market, Health, and Population. It uses deterministic synthetic data to practice the `Reproduce -> Diagnose -> Transfer` workflow without claiming access to proprietary survey, claims, employer, treatment, or employee-assistance records.

Primary anchor: Bubonya, Cobb-Clark, and Wooden on mental health and productivity at work.

Challenge anchor: job meaning, identity, and employee mental health.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes compact outputs under `output/reproduced/`, `output/diagnosed/`, and `output/transfer/`.
