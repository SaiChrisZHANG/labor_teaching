# Week 1 Lab: Health Measures And Labor Supply Identification

This lab is the bounded Week 1 teaching path for Labor Market, Health, and Population. It uses deterministic synthetic data to practice the `Reproduce -> Diagnose -> Transfer` workflow without confidential health, retirement, disability, or administrative microdata.

Primary anchor: Blundell, Britton, Costa Dias, and French on health measures and employment near retirement.

Challenge anchor: Bound on self-reported versus objective health measures.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes compact outputs under `output/reproduced/`, `output/diagnosed/`, and `output/transfer/`.
