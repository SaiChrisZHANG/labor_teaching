# Week 3 Lab: Fertility, Mortality, Caregiving, And Lifecycle Labor Allocation

This lab is the bounded Week 3 teaching path for Labor Market, Health, and Population. It uses deterministic synthetic data to practice the `Reproduce -> Diagnose -> Transfer` workflow without confidential health, family, pension, care, survey, or administrative records.

Primary anchor: Fadlon and Nielsen on family labor-supply responses to severe health shocks.

Challenge anchor: Ameriks and coauthors on older-worker flexibility and late-life job design.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes compact outputs under `output/reproduced/`, `output/diagnosed/`, and `output/transfer/`.
