# Week 2 Lab: Disability, Chronic Conditions, Labor Supply, And Job Choice

This lab is the bounded Week 2 teaching path for Labor Market, Health, and Population. It uses deterministic synthetic data to practice the `Reproduce -> Diagnose -> Transfer` workflow without confidential disability, health, benefit, survey, or employer records.

Primary anchor: Meyer and Mok on disability, earnings, income, and consumption.

Challenge anchor: Hill, Maestas, and Mullen on employer accommodation and labor supply.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes compact outputs under `output/reproduced/`, `output/diagnosed/`, and `output/transfer/`.
