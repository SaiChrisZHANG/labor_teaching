# Original And Reduced Data Notes

No official replication package is bundled with this lab. The smoke path creates deterministic synthetic teaching data in `original/reduced/disability_chronic_conditions_synthetic.csv`.

The synthetic file is designed only to illustrate event-study timing, accommodation, program interaction, local demand, and selection mechanisms in the spirit of Meyer and Mok and Hill, Maestas, and Mullen. It should not be interpreted as empirical evidence.
