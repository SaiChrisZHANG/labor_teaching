## Lifecycle labor-allocation framework

| Object | Labor meaning | Key state variables | Main margins |
|---|---|---|---|
| Fertility timing | Arrival of care and time-allocation shocks | age, parity, children’s ages, partner income | participation, hours, occupation, mobility |
| Mortality risk / severe illness | Resource loss + care demand + changed horizon | own/spouse/parent health, expected survival, insurance | hours, participation, savings, retirement |
| Caregiving | Informal care crowds out market work and raises flexibility demand | care recipient status, intensity, formal-care access | hours, retirement timing, occupation, location |
| Aging | Changes work capacity and value of leisure/flexibility | age, health, pension eligibility, wealth | late-life labor supply, retirement, bridge jobs |
| Pensions / retirement savings | Shift the budget set and timing incentives | eligibility ages, wealth, benefits, taxes | claiming, exit timing, partial retirement, reentry |
