# Original / Reduced Teaching Data

This folder stores deterministic synthetic data for the Week 3 reproduce and diagnose exercises.

The files are created by:

```bash
python src/make_synthetic_data.py
```

Generated files:

- `reduced/card_krueger_synthetic.csv`
- `reduced/staggered_minimum_wage_synthetic.csv`

These are not official replication files. They are bounded teaching datasets designed to make the DID, event-study, modern DID, and clustering decisions executable without external data.
