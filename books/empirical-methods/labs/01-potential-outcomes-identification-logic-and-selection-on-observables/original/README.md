# Original Anchor Notes

This lab is inspired by the LaLonde and Dehejia-Wahba training-program sequence. The local teaching path does not ship the official National Supported Work data or claim to reproduce published magnitudes.

The reduced folder contains a deterministic synthetic dataset with variables patterned after the canonical training-program setup:

- treatment status;
- age, education, race and ethnicity indicators, marital status, and degree status;
- pre-treatment earnings;
- post-treatment earnings.

Use this folder to keep notes on official replication-package paths if an instructor later adds the original data.
