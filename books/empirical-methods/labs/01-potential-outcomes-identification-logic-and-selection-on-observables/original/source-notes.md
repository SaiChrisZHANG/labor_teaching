# Source Notes

The intellectual anchor is LaLonde's comparison of experimental and nonexperimental training-program evaluations, followed by Dehejia and Wahba's propensity-score matching reanalysis.

This course folder uses a bounded synthetic path for three reasons:

1. the lab should run without external downloads;
2. students should focus first on identification logic rather than official package management;
3. the synthetic data can make overlap, balance, and selection visible in a compact classroom setting.

The synthetic file is generated by `src/make_synthetic_data.py` and written to `original/reduced/training_synthetic.csv`. It should be interpreted as a pedagogical analog, not as evidence about the actual National Supported Work program.
