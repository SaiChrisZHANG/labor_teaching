# Transfer data notes

The transfer file shifts the source task away from job postings and toward service-record text. The labels now describe flexible service capacity rather than remote-work feasibility.

The source text classifier is deliberately imperfect in this setting. Some transfer records use terms such as `virtual intake`, `tele-service`, `mobile casework`, and `digital queue`, which partly overlap with source remote-work language but carry a different institutional meaning. The image score is included to mimic a second unstructured or semi-structured proxy that may add signal while creating a new validation burden.

The transfer exercise asks students to decide whether the source benchmark travels, what should be relabeled, and how subgroup performance changes before any downstream economics claim is made.
