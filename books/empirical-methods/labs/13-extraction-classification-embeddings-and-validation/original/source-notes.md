# Source notes

The reduced source file is a deterministic synthetic job-posting dataset inspired by applied economics work that validates text-derived labor-market measures [@gentzkowKellyTaddy2019; @hansenRemoteWorkAcross2023].

The file is not paper data and should not be cited as an empirical estimate. It is designed to teach:

- annotation and adjudication;
- inter-rater reliability;
- dictionary, embedding-style, and supervised validation;
- confusion matrices and calibration;
- subgroup performance;
- downstream measurement-error consequences.

The dataset includes a simulation-truth label only because this is a teaching lab. In real research, the latent economic object is not directly observed, so the benchmark and validation evidence become central.
