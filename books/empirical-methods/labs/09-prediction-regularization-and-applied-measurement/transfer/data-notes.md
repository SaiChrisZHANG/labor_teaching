# Transfer Data Notes

The transfer file is a deterministic synthetic occupation-title dataset inspired by machine-learning occupational classification work [@ikudoOccupationalClassificationsMachine2018].

It is not an official replication file. It is designed to test whether a model trained on richer job-posting text can be reused for terse administrative job titles.

The transfer target is `technology_occupation`. The exercise asks students to evaluate vocabulary coverage, calibration, subgroup performance, and confusion-matrix evidence before using predicted titles as downstream economic measures.
