# Original Teaching Inputs

This folder contains reduced synthetic inputs for the Week 17 spatial structural modeling lab.

The files are created by:

```bash
python src/make_synthetic_data.py
```

## Files

- `reduced/locations_synthetic.csv`: synthetic locations with productivity, amenities, housing supply, job capacity, and baseline equilibrium objects.
- `reduced/bilateral_commuting_synthetic.csv`: synthetic residence-workplace commuting flows and bilateral commute costs.

These files are not an official replication package for any published paper.
