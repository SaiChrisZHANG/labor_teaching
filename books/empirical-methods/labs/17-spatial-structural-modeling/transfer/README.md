# Transfer Teaching Inputs

This folder contains the transfer inputs for the Week 17 dynamic spatial adjustment exercise.

The files are created by:

```bash
python src/make_synthetic_data.py
```

## Files

- `data/dynamic_policy_synthetic.csv`: a synthetic productivity shock and adjustment speeds used to compare impact, medium-run, and long-run outcomes.

The transfer task is inspired by dynamic urban economics but does not solve a full dynamic model.
