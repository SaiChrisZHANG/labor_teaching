# Original / reduced data

The bounded teaching path uses deterministic synthetic data generated by `src/make_synthetic_data.py`.

The generated file is:

- `reduced/compulsory_schooling_synthetic.csv`

It is modeled on compulsory-schooling IV logic: a rule-based exposure shifts the probability of staying in school, and the outcome is an earnings measure. It does not reproduce official Angrist-Krueger or Oreopoulos data or published magnitudes.
