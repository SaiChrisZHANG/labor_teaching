# Original teaching path

The original path uses deterministic synthetic city data to reproduce the logic of a Hsieh-Moretti-style spatial-equilibrium counterfactual.

The data are stored in:

```text
original/reduced/spatial_equilibrium_synthetic.csv
```

They contain city-level productivity, housing capacity, housing-supply elasticity, amenity values, baseline wages, rents, population, and output. The goal is to teach the equilibrium counterfactual architecture, not to reproduce published magnitudes.
