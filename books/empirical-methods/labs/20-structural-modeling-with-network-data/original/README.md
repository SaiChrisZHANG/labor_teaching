# Original Teaching Path

This folder contains the reduced synthetic teaching data for the Week 20 reproduce and diagnose tasks.

The files are generated by:

```bash
python src/make_synthetic_data.py
```

Expected generated files:

- `reduced/workers_synthetic.csv`
- `reduced/dyad_opportunities_synthetic.csv`
- `reduced/network_edges_synthetic.csv`

These files are not official replication data. They are deterministic teaching data used to demonstrate how a structural network workflow connects link formation, behavior on networks, moments, validation, and counterfactuals.
