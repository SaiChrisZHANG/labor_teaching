# Transfer Teaching Path

This folder contains the synthetic referral-search data used for the Week 20 transfer exercise.

The file is generated by:

```bash
python src/make_synthetic_data.py
```

Expected generated file:

- `data/referral_search_synthetic.csv`

The transfer task asks students to compare a policy that improves information quality on existing referral links with a policy that adds cross-group referral opportunities.
