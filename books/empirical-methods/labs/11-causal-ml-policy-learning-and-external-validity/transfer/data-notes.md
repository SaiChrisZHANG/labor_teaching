# Transfer Data Notes

The target file is a deterministic synthetic dataset with a different population mix and a weaker implementation environment than the source file. It is designed to make external-validity questions concrete:

- the target population has more rural applicants, higher local unemployment, lower average digital skill, and longer unemployment spells;
- the treatment technology is attenuated in the target setting, especially in rural areas;
- the source-trained rule can be applied to the target covariates, but this does not guarantee target value.

The exercise is inspired by external-validity and transportability papers such as Allcott, Vivalt, and Dehejia, Pop-Eleches, and Samii [@Allcott2015; @Vivalt2020; @DehejiaPopElechesSamii2021]. It is not an official replication and should not be described as one.
