# Transfer Data Notes

The transfer data are inspired by referral experiments and referral-network inequality research [@pallaisEvidenceFieldExperiments2016; @barwickJobReferralNetwork2019].

Each row is an employee-applicant referral dyad. The row contains referrer characteristics, applicant characteristics, tie strength, same-group status, and a synthetic callback outcome.

The teaching purpose is to show why pair-level data require pair-aware inference. Dyads sharing the same referrer or the same applicant are not independent rows.
