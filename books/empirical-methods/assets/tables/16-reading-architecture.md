# Reading Architecture

| Role in the lecture | Paper(s) | What students should learn |
|---|---|---|
| Spatial inference benchmark | [@conley1999] | What spatially robust inference does and does not fix |
| Interference / exposure mapping anchor | [@miguelKremer2004] | Why SUTVA fails and how externalities can be measured rather than ignored |
| Border / spatial RD anchor | [@black1999; @dell2010] | How signed-distance and boundary logic create local identification |
| Contiguous-border design anchor | [@dubeLesterReich2010] | How to exploit nearby comparisons while still worrying about local shocks |
| Shift-share design anchor | [@goldsmithPinkhamSorkinSwift2020; @borusyakHullJaravel2025] | How modern shift-share designs are evaluated and diagnosed |
| Place-based / geography-choice anchor | [@bussoGregoryKline2013] | Why geography and local equilibrium matter for interpretation |
| Sorting / mover-design anchor | [@chettyHendrenKatz2016] | When place is the treatment and sorting is the main threat |
