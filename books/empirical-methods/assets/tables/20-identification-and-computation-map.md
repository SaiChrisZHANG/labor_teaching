# Identification and Computation in Structural Network Models

| Problem | Core question | Typical solution | Main caveat |
|---|---|---|---|
| Endogenous links | Are outcomes driving links or links driving outcomes? | Structural formation equation, timing restrictions, panel network variation | Strong assumptions about link utility and equilibrium |
| Latent heterogeneity | Are hubs / homophily / degree patterns observed or latent? | Degree heterogeneity, random effects, latent-type models | Can absorb too much if not disciplined by moments |
| Strategic interaction | Are peers responding simultaneously? | Equilibrium action model, fixed point / best response system | Reflection-like identification issues remain if exposure is weak |
| High-dimensional network state | How do we estimate when the network is huge? | Moment selection, sparse approximations, simulation, variational methods | Computation may dominate identification logic |
| Counterfactual policy | What changes under a link/information intervention? | Simulate policy under estimated formation + behavior system | Policy relevance depends on correct formation assumptions |
