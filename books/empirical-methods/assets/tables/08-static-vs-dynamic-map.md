:::{table} Static, partial-equilibrium, and equilibrium structure
:label: tbl:static-vs-eq
| Model type | Typical object | What is latent | What policy/counterfactuals it helps answer |
|---|---|---|---|
| Static reduced form | Average treatment effect | Counterfactual untreated outcome | Local intervention effects |
| Static structural | Preferences/technology | Utility/cost primitives | Counterfactuals under unobserved rules or prices |
| Dynamic structural | State dependence / expectations | Value functions, state transitions | Dynamic incentives, forward-looking behavior |
| Search equilibrium | Tightness, vacancies, wages | Meeting technology, surplus division | Vacancy responses, unemployment, wage incidence |
| Spatial equilibrium | Location utility, rents, migration/commuting | Mobility frictions, amenity value, housing incidence | Place-based policy, housing constraints, migration response |
| Market equilibrium | Prices, quantities, markups, entry | Demand substitution, cost, conduct | Pass-through, market power, welfare, merger/policy incidence |
| Flow / gravity equilibrium | Bilateral flows across units | Trade/migration/commuting frictions | Reallocation across places or markets |
:::
