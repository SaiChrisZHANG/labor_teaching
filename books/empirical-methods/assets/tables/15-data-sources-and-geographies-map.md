# Data Sources and Geographies Map

| Data source | Typical spatial unit | Typical labor use | Common challenge |
|---|---|---|---|
| ACS / Census | tract, county, PUMA, state | local wages, commuting, demographics, migration | changing boundaries / harmonization |
| LEHD / LODES / QWI | workplace/residence block, tract, county | job access, worker flows, workplace geography | disclosure control, limited variables |
| Opportunity Atlas / tract-level linked admin data | tract / county | neighborhood exposure, intergenerational mobility | exposure timing and tract stability |
| IRS migration / address histories | county, state, commuting zone | migration responses to shocks, place mobility | limited covariates, coarse location |
| Zillow / FHFA / CoreLogic | ZIP, tract, county | rents, house prices, capitalization, incidence | nonuniform coverage / changing geography |
| Transit / routing APIs / GTFS | network, stop, route, travel-time matrix | job access, commute burdens, modal feasibility | reproducibility and time variation |
| Remote sensing / rasters | pixel/grid | pollution, heat, land use, lights | resolution and aggregation |
| Historical maps / shapefiles | digitized historical units | path dependence, spatial discontinuities | georeferencing and boundary comparability |
