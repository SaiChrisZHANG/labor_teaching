## Spatial structural diagnostics

| Issue | Key question | Why it matters | Typical failure mode |
|---|---|---|---|
| Equilibrium object | What is the actual object of interest? | Prevents unnecessary structure | Modeling wages only when welfare/incidence is the target |
| Adjustment margins | Which margins adjust? | Determines incidence | Ignoring commuting, rents, or migration |
| Identification | Which data pin down which primitives? | Keeps the model interpretable | Treating calibrated parameters as estimated facts |
| Fit | Which moments are targeted vs untargeted? | Prevents overclaiming | Good fit on targeted moments only |
| Counterfactual distance | How far outside observed support is the policy? | Shapes credibility | Large extrapolation with weak validation |
| Welfare | What surplus or utility object is reported? | Makes results economically interpretable | Using nominal wage changes as welfare |
