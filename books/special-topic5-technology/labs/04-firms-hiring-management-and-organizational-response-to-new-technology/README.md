# Week 4 Lab: Firms, Hiring, Management, And Organizational Response To New Technology

This lab is the bounded Week 4 teaching path for Technology, Innovation, and Labor. It uses deterministic synthetic data to practice the `Reproduce -> Diagnose -> Transfer` workflow without official firm resume data, job-posting microdata, confidential employer-employee records, vendor telemetry, or external downloads.

Primary anchor: Babina, Fedyk, He, and Hodson's firm AI-investment and workforce-composition design.

Challenge transfer: algorithm aversion, worker and manager AI attitudes, and principal-agent tensions in hiring, screening, monitoring, and evaluation.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes compact outputs under `output/reproduced/`, `output/diagnosed/`, and `output/transfer/`.
