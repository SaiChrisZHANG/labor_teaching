# Week 6 Lab: Synthesis, Frontier Questions, And Student Research Designs

This lab is the bounded Week 6 teaching path for Technology, Innovation, and Labor. It uses deterministic synthetic data to practice the `Reproduce -> Diagnose -> Transfer` workflow without official robot data, Census microdata, firm-adoption data, platform logs, worker telemetry, or external downloads.

Primary anchor: Acemoglu and Restrepo's local robot-exposure design.

Challenge transfer: generative AI worker use, firm adoption, organizational implementation, platform governance, compute infrastructure, and global service trade.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes compact outputs under `output/reproduced/`, `output/diagnosed/`, and `output/transfer/`.
