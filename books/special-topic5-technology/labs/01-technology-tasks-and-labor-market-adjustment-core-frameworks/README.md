# Week 1 Lab: Technology Measures, Tasks, And Labor Exposure

This lab is the bounded Week 1 teaching path for Technology, Innovation, and Labor. It uses deterministic synthetic data to practice the `Reproduce -> Diagnose -> Transfer` workflow without confidential patent, resume, firm, vacancy, or worker microdata.

Primary anchor: Webb's task and patent-text exposure logic.

Measurement contrast: Kogan, Papanikolaou, Schmidt, and Seegmiller's worker-level technology exposure logic.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes compact outputs under `output/reproduced/`, `output/diagnosed/`, and `output/transfer/`.
