# Week 5 Lab: Technology, Inequality, Market Structure, And Labor-Market Institutions

This lab is the bounded Week 5 teaching path for Technology, Innovation, and Labor. It uses deterministic synthetic data to practice the `Reproduce -> Diagnose -> Transfer` workflow without confidential firm microdata, official Economic Census files, establishment technology surveys, data-center vendor files, or external downloads.

Primary anchor: Autor, Dorn, Katz, Patterson, and Van Reenen's superstar-firm and labor-share design.

Challenge transfer: structural change within versus across firms, Global South technology diffusion, outsourcing, and frontier AI infrastructure incidence.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes compact outputs under `output/reproduced/`, `output/diagnosed/`, and `output/transfer/`.
