# Week 2 Lab: Automation, AI, And Labor Demand

This lab is the bounded Week 2 teaching path for Technology, Innovation, and Labor. It uses deterministic synthetic data to practice the `Reproduce -> Diagnose -> Transfer` workflow without official robot data, confidential firm records, proprietary vacancy data, or external downloads.

Primary anchor: Acemoglu and Restrepo's local robot exposure design.

Challenge paper: Aghion, Bergeaud, Boppart, Klenow, and Li on firm-level AI use and labor demand.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes compact outputs under `output/reproduced/`, `output/diagnosed/`, and `output/transfer/`.
