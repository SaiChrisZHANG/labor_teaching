# Week 3 Lab: Worker Adjustment, Skills, Training, And Career Transitions

This lab is the bounded Week 3 teaching path for Technology, Innovation, and Labor. It uses deterministic synthetic data to practice the `Reproduce -> Diagnose -> Transfer` workflow without official patent-worker data, confidential administrative records, firm training microdata, or external downloads.

Primary anchor: Kogan, Papanikolaou, Schmidt, and Seegmiller's worker-level labor-displacing technology exposure logic.

Challenge transfer: training and retirement responses under local exposure, plus incumbent versus new-hire training under AI adoption.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes compact outputs under `output/reproduced/`, `output/diagnosed/`, and `output/transfer/`.
