# Source Notes

The synthetic inherited-culture file is inspired by the design logic in [@fernandezFogli2009]. It includes origin norm proxies, current peer pressure, host labor-demand and wage controls, childcare costs, family structure, and labor-supply outcomes.

The synthetic job-entry file is inspired by the field-experimental logic in [@floryLeibbrandtList2015] and the gender-coded entry question in [@delfino2024]. It varies job framing, perceived norm fit, and application behavior.

These files are teaching data only. They are not official replication data.
