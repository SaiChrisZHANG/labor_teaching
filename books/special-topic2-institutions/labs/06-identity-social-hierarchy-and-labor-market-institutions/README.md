# Week 6 Lab: Identity, Social Hierarchy, and Labor-Market Institutions

This lab is the bounded Week 6 teaching path for Special Topic 2. It uses deterministic synthetic data to practice the Reproduce -> Diagnose -> Transfer workflow without confidential audit-study, immigrant-resume, caste-network, contact-experiment, public personnel, or structural-estimation data.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```
