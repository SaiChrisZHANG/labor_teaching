
:::{table} Common Week 13 proposal failure modes and fixes
:label: tbl:week13-failure-modes
| Failure mode | Symptom | Fix |
| --- | --- | --- |
| Topic without a question | "I want to study migration" | State a precise margin, population, and outcome |
| Mechanism without design | Story is plausible but not testable | Specify variation that isolates the mechanism |
| Design without labor contribution | Nice identification but generic | Explain why the question changes labor theory, policy, or allocation |
| Too many outcomes | Project reads like a dashboard | Choose one primary estimand and a small set of complements |
| Hidden equilibrium issue | Worker-side design ignores firms or prices | State whether the project is partial equilibrium or needs Labor II tools |
| Literature survey posing as idea | Memo lists papers but no contribution | Add one sentence that starts with "This paper asks..." |
