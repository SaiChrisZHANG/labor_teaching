:::{table} Empirical toolkit map for inequality analysis
:label: tbl:ineq-toolkit-map
| Method | Main use | Data requirement | Main limitation |
| --- | --- | --- | --- |
| Percentile-series description | Track upper-tail, middle, and lower-tail changes over time | Repeated cross-sections or panels with consistent pay measures | Descriptive only; mechanism is not identified |
| Juhn--Murphy--Pierce style residual decomposition | Separate observed-characteristic changes from residual inequality | Rich micro data with stable covariates | Residual mixes many mechanisms and measurement issues |
| DiNardo--Fortin--Lemieux counterfactuals | Construct reweighted distributional counterfactuals for composition/institution changes | Repeated cross-sections with overlap | Requires support/overlap and remains partially descriptive |
| RIF / unconditional quantile methods | Estimate how covariates shift different points of the outcome distribution | Micro data and influence-function implementation | Interpretation depends on chosen functional and reweighting assumptions |
| AKM linked employer--employee decomposition | Partition wages into worker effects, firm premia, and sorting | Linked worker-firm panel data | Requires strong mobility/connected-set structure and can be sensitive to design choices |
:::
