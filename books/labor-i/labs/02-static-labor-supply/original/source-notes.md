# Source notes

## Verified source paper

- **Article:** Saez, Emmanuel. 2010. *Do Taxpayers Bunch at Kink Points?* *American Economic Journal: Economic Policy* 2(3): 180--212.
- **Official replication source:** AEA data and code archive for `@saez2010`

## Why this paper fits Week 2

This paper is ideal for the static labor-supply module because it turns nonlinear budget-set geometry into an observable empirical object. The chapter's EITC-style schedule and the lab's bunching graph are different views of the same Week 2 problem: local incentives under a kinked schedule.

## Teaching note

The bounded teaching path included here is intentionally reduced. It preserves the logic of binning earnings near a kink and inspecting local mass, but it does not attempt to recreate the paper's exact data construction. For a closer replication, students still need the official package files and any underlying microdata objects referenced there.
