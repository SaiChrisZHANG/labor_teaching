# Transfer folder README

This folder stores the Week 9 transfer inputs and notes.

- `data-notes.md` explains how the synthetic employer file maps to the challenge anchor.
- `data/employer_report_card_synthetic.csv` is generated locally by `src/build_week9_synthetic_data.py`.

The transfer path is designed to move from one average callback effect to employer-level discrimination measurement without requiring students to run a real field experiment.
