# Transfer folder README

This folder stores the Week 7 transfer inputs and notes.

- `data-notes.md` explains how the synthetic working-conditions file maps to the challenge anchor.
- `data/maestas_working_conditions_synthetic.csv` is generated locally by `src/build_week7_synthetic_data.py`.

The transfer path is designed to move from explicit amenity valuation to broader working-conditions bundles and value-adjusted inequality accounting.
