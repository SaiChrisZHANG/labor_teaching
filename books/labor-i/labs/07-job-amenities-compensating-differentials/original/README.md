# Original folder README

This folder stores the bounded Week 7 reproduction inputs and notes.

- `source-notes.md` explains the paper benchmark and what the synthetic file preserves.
- `reduced/mas_pallais_job_choice_synthetic.csv` is generated locally by `src/build_week7_synthetic_data.py`.

The synthetic file is built for teaching willingness to pay for explicit job amenities. It is not a literal release of the original paper's underlying data.
