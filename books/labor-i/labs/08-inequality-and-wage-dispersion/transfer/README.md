# Transfer folder README

This folder stores the Week 8 transfer inputs and notes.

- `data-notes.md` explains how the synthetic worker-firm file maps to the challenge anchors.
- `data/worker_firm_inequality_synthetic.csv` is generated locally by `src/build_week8_synthetic_data.py`.

The transfer path is designed to move from repeated-cross-section inequality objects to worker-firm decomposition logic without requiring restricted administrative microdata.
