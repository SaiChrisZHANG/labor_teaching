# Transfer exercise notes

Start by running the factbook script on `data/labor_micro_synthetic.csv`, then replace that file with a real bounded dataset or extension sample.

## Required columns for the default script

- `date`
- `employment_status` with values in `{employed, unemployed, nilf}`
- `weight`
- `hours`
- `weekly_earnings`

Optional but recommended:

- `group`
- `sex`
- `education`
- `age_band`
- `state`

## Suggested transfer choices

- one demographic subgroup from a public labor extract;
- one later period from the same data source;
- one local labor-market panel with clearly documented provenance.

Keep the extension small enough to yield one clean figure and one short interpretation note.
