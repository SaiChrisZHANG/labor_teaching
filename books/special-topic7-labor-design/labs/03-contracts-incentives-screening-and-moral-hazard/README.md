# Week 3 Lab: Contracts, Incentives, Screening, And Moral Hazard

This lab is the bounded Week 3 teaching path for Labor Market Design, Contracting, and Mechanism Design. It uses deterministic synthetic worker-contract data to practice the `Reproduce -> Diagnose -> Transfer` workflow without claiming access to proprietary personnel files, payroll records, supervisor-rating data, confidential workplace experiments, or official replication packages.

Primary anchor: Lazear on performance pay and productivity.

Challenge anchor: de Janvry, He, Sadoulet, Wang, and Zhang on subjective performance evaluation and influence activities, with an optional commitment path inspired by Kaur, Kremer, and Mullainathan.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes compact outputs under `original/reduced/`, `output/reproduced/`, `output/diagnosed/`, and `output/transfer/`.
