# Week 1 Lab: Matching, Market Design, And Labor Allocation

This lab is the bounded Week 1 teaching path for Labor Market Design, Contracting, and Mechanism Design. It uses deterministic synthetic applicant-program data to practice the `Reproduce -> Diagnose -> Transfer` workflow without claiming access to official NRMP rank-order lists, confidential applicant records, program files, or replication packages.

Primary anchor: Roth and Peranson on the redesign of the medical residency match.

Challenge anchor: Agarwal on empirical and policy analysis in the medical match.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes compact outputs under `original/reduced/`, `output/reproduced/`, `output/diagnosed/`, and `output/transfer/`.
