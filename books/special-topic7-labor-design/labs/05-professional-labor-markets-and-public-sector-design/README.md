# Week 5 Lab: Professional Labor Markets And Public-Sector Design

This lab is the bounded Week 5 teaching path for Labor Market Design, Contracting, and Mechanism Design. It uses deterministic synthetic structured labor-market data to practice the `Reproduce -> Diagnose -> Transfer` workflow without claiming access to confidential Army, school, public-service, civil-service, or personnel records.

Primary anchor: Davis, Greenberg, and Jones on deferred acceptance in Army officer labor markets.

Challenge anchor: Davis on matching Teach For America teachers to schools, with extensions to internal talent markets, public-service recruitment, and public-sector entry/exit margins.

Run the smoke path from this directory:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes compact outputs under `original/reduced/`, `output/reproduced/`, `output/diagnosed/`, and `output/transfer/`.
