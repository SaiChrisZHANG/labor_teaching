# Theory-to-empirical bridge

| Theory object | Observable labor margin | Common design | Representative applied question |
|---|---|---|---|
| Hidden effort | Output per hour, sales, production, attendance | Contract change / field experiment | Did a performance-pay reform raise effort or only sorting? |
| Hidden type | Entry, retention, wage progression, probation success | Menu variation / promotion threshold / matched panel | Which workers sort into incentive pay or survive probation? |
| Commitment | Contract take-up, schedule adherence, target completion | Randomized contract framing / menu experiments | Do workers demand commitment-enhancing work arrangements? |
| Multitasking | Task mix, quality vs quantity, gaming | Change in measured targets / subjective-to-objective shift | Does the contract distort effort toward measured tasks? |
| Subjective evaluation | Ratings, promotions, evaluator-specific work allocation | Random evaluator assignment / evaluator uncertainty | Do workers redirect effort toward pleasing specific evaluators? |
| Social / relational motives | Reciprocity, response to gifts, peer/team outcomes | Field gifts / bonus variation / team incentives | Are incentives amplified or crowded out by social preferences? |
