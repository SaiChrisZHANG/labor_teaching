# Principal-agent map for labor contracts

| Object | Canonical labor question | Hidden or observed? | Typical labor margin |
|---|---|---|---|
| Effort | How does pay affect performance? | Hidden action | Output, attendance, speed, quality |
| Type | Which workers sort into which contracts? | Partly hidden | Entry, quits, probation success, wage growth |
| Commitment / self-control | Do workers want discipline-enhancing arrangements? | Hidden preference / dynamic inconsistency | Contract choice, attendance, target completion |
| Multitasking | What happens when only some outputs are measured? | Hidden task allocation | Task mix, gaming, neglect of soft tasks |
| Subjective evaluation | What is rewarded when supervisors rate workers? | Signal quality problem | Promotions, ratings, evaluator-specific effort |
| Relational incentives | When do informal promises substitute for formal incentives? | Hidden enforcement environment | Retention, discretionary bonuses, effort persistence |
