## Core design concepts map

| Design object | What is designed? | Labor-market margin | Main concern |
|---|---|---|---|
| Visibility / ranking | Which workers or jobs are shown first | Applications, interviews, hire rates | Allocation, bias, visibility inequality |
| Assignment / routing | Which worker is matched to which task or client | Match quality, queue length, fill rates | Efficiency, congestion, fairness |
| Wage / price rule | Posted wage, bids, floors, commissions, surge multipliers | Wage offers, participation, rent capture | Incidence, bargaining power, risk |
| Information / reputation | Ratings, histories, guarantees, salary disclosure | Search targeting, employer beliefs, wage bids | Trust, learning, stratification |
| Governance / dispute rules | Appeals, deactivation, refunds, penalties | Risk exposure, continuity, platform dependence | Power, discipline, worker welfare |
