## Data sources map

| Data source | What it is good for | Example use |
|---|---|---|
| ACS / PUMS / Decennial Census | Migration, commuting, demographics, rents | Track who moves after local shocks |
| LEHD / LODES / QWI | Worker-firm-place links, commuting, job flows | Separate commuting from migration adjustment |
| IRS county-to-county migration | Inter-county migration flows | Predicted-destination IVs and migration incidence |
| Zillow / CoreLogic / FHFA / ACS rents | Housing capitalization | Incidence on landlords vs workers |
| Building permits / construction employment | Housing supply and demand response | Migration accelerator / local boom channels |
| Tract-level employment or business directories | Neighborhood job composition | Gentrification as labor-market restructuring |
| Transit / travel-time matrices | Commuting accessibility | Estimate commuting openness and access changes |
| Matched employer-employee administrative data | Worker outcomes by origin, destination, and firm | Persistence, worker scarring, heterogeneity |
| Historical settlement shares / origin-destination matrices | Migration design inputs | Shift-share / destination-prediction instruments |
