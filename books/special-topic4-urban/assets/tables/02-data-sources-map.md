# Data sources map

| Data source | What it is good for | Typical labor-housing use |
|---|---|---|
| ACS / ACS PUMS | Migration, tenure, rent, commute time, worker characteristics | Real access, housing burden, migration heterogeneity, commuting by worker type |
| Decennial Census / Census long form | Long-run spatial population and housing structure | Historical adjustment, long-run sorting, persistent place differences |
| LEHD / LODES / QWI | Worker-job location linkages, employment counts, workplace vs residence | Commuting flows, within-metro access, local employment adjustment |
| IRS migration data | Inter-county/state migration flows | Mobility response after local shocks, migration heterogeneity |
| Zillow / FHFA / CoreLogic / transaction-level housing data | Prices, rents, turnover, capitalization, house values | Rent capitalization, incidence, owner vs renter exposure |
| HUD administrative data / voucher data | Subsidized housing, program participation, mobility supports | Housing assistance and labor-market access where program access is relevant |
| HMDA | Mortgage application and approval data | Credit constraints, ownership access, financing barriers where homeownership is relevant |
| Commuting-zone / metro / tract / ZIP delineations | Geographic market definition | Sensitivity of results to labor-market and housing-market delineation |
| Local housing-supply / regulation measures | Elasticities, land-use restrictions, permitting | Identify supply-constrained places and heterogeneity in capitalization |
