| Element | Notation / idea | Interpretation |
|---|---|---|
| Job | `{math}`j`` | Workplace and wage offer |
| Residence | `{math}`r`` | Housing location and rent bundle |
| Wage | `{math}`w_j`` | Earnings associated with the workplace |
| Rent / housing cost | `{math}`R_r`` | Residential cost of accessing place `{math}`r`` |

| Amenities | `{math}`A_r`` | Local nonwage components of welfare |
| Idiosyncratic fit | `{math}`\varepsilon_{ijr}`` | Worker-specific taste or match value |
| Feasible set | `{math}`\mathcal{J}_i(c)`` | Jobs reachable within a generalized commuting-cost threshold |
| Spatial equilibrium | `{math}`w_\ell - R_\ell + A_\ell = \bar U`` | Wages, rents, and amenities jointly allocate workers across places |
