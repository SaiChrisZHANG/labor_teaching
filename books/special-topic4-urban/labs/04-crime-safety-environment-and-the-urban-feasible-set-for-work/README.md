# Week 4 Lab

This lab is a bounded synthetic workflow for Week 4 of Urban and Labor.

It anchors the reproduce step to the displacement-to-crime design in Khanna, Medina, Nyshadham, and Tamayo. It does not use confidential administrative data or reproduce official estimates.

Run the smoke path from this folder:

```bash
ENV_NAME=research bash smoke.sh
```

The script writes deterministic outputs under `output/`.
